This document outlines some of the user stories for the BlooBase platform, a local artisan marketplace. These stories help define the key features and functionalities from a user-centered perspective.

## User Stories

### 1. User Login via Third-Party Account

As a user, I want to log in using my Google or Microsoft account so that I can access the platform quickly and securely without creating a new account.

### 2. Artisan Profile Setup

> As an artisan, I want to set up my profile so that customers can learn more about me and my offerings.

### 3. Product Photo Upload

> As an artisan, I want to upload pictures of my products so that users can see what I’m selling before making a purchase.

### 4. Product Search

> As a user, I want to search for products on the platform so that I can find items that match my needs and preferences.

### 5. Order Tracking

> As a user, I want to track the status of my order so that I know when to expect delivery or completion of the service.

### 6. Earnings Dashboard for Artisans

> As an artisan, I want to view a dashboard of my sales and earnings so that I can track my business performance over time.

### 7. Cart

> As a user, I want to save products or artisans to a cart so that I can easily find and purchase them later.

### 8. Secure Payment Integration

> As a user, I want to pay securely through the platform so that I feel confident my payment information is safe.

### 9.Store verification/removal

> As an admin ,I want to view all the stores created on the platform and I should be able to remove and verify stores so I can ensure integrity and order on the platform

### 10.Inventory Tracking

> As an artisan,I want to keep track of my inventory/stock on the platform so I can know when to restock my products.
